# PrintBridge

**Turn any printer into an AirPrint printer.**

PrintBridge is a Windows app that makes any Windows-connected printer available to iPhones, iPads, and Macs via AirPrint — even old printers without built-in AirPrint support.

## Download

Get the latest version from the [Releases](https://github.com/SpicyCoffee-One/PrintBridge-releases/releases) page.

**System requirements:** Windows 10 or later (64-bit)

## Features

- Share any Windows printer as an AirPrint-compatible printer
- Apple devices discover shared printers automatically — nothing to install
- Active Directory (LDAP) and Entra ID (Azure AD) authentication
- Audit logging with CSV export
- Runs as a Windows service — starts with Windows, no login required
- No cloud, no subscriptions — everything stays on your local network
